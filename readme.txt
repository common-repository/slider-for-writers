== Installation ==

It's easy to install.

1. Extract `slider-for-writers.zip` and upload the folder `slider-for-writers` to the `/wp-content/plugins/` directory;
2. Activate the plugin through the `Plugins` menu in WordPress;
3. Configure plugin
